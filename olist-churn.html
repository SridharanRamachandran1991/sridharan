Olist Ecommerce Churn Analysis

Comprehensive analysis of customer churn patterns in Brazilian e-commerce

Project Overview

This project presents an in-depth analysis of customer churn patterns in the Olist e-commerce platform, one of Brazil's largest e-commerce marketplaces. The analysis explores customer behavior, identifies key churn indicators, and provides actionable insights through comprehensive data visualization.

Key Objectives

  • Identify patterns and trends in customer churn behavior
  • Analyze customer segmentation and purchase patterns
  • Discover predictive indicators of customer attrition
  • Provide actionable insights to reduce churn and improve retention strategies
  • Visualize complex data relationships through interactive dashboards

Dataset

The analysis is based on the Brazilian E-Commerce Public Dataset by Olist, which contains information about orders made at Olist Store. The dataset includes multiple CSV files with data on orders, customers, products, sellers, reviews, and geolocation.

View Dataset on Kaggle

Methodology

The project employs advanced data analysis techniques including exploratory data analysis (EDA), customer segmentation, time series analysis, and predictive modeling. The findings are presented through interactive Tableau dashboards that allow stakeholders to explore the data dynamically and gain insights into customer behavior patterns.
